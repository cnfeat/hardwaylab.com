# 笨方法实验室

> 世上无难事，只怕笨方法。

笨方法实验室是一个致力于帮助普通人从0到1构建长期主义人生成长系统的在线学习平台。

## 🎯 项目简介

笨方法实验室提供系统化的课程内容，帮助学习者通过"笨方法"实现持续成长和自我提升。

## 📚 课程体系

| 课程 | 简介 |
|------|------|
| **笨方法文化手册** | 人生成长系统的核心框架，27万字行动指南 |
| **卡片创作100讲** | 卡片写作精华实践之法，100天构建创作系统 |
| **个人生产力发展指南** | 产品货架、身份组合、生产方式三角理论 |
| **日课一问** | 每日三省破局人生，400+灵魂拷问 |

## 🔧 技术栈

- **框架**: Jekyll 4.4.1
- **主题**: Just the Docs
- **语言**: Markdown / Liquid
- **部署**: GitHub Pages

## 🚀 快速开始

### 安装依赖

```bash
bundle install
```

### 本地开发

```bash
bundle exec jekyll serve --no-watch
```

访问 http://localhost:4000 查看网站。

### 构建生产版本

```bash
bundle exec jekyll build
```

生成的文件在 `_site` 目录中。
